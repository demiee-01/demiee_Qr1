# Demlee QR Generator

A clean, glassmorphism-style QR code generator built with React, Vite, and Tailwind CSS.

---

## Clone the repo

**HTTPS**
```bash
git clone https://github.com/demiee-01/demiee_Qr1.git
```

**SSH**
```bash
git clone git@github.com:demiee-01/demiee_Qr1.git
```

---

## Setup & Run

```bash
cd demiee_Qr1/qr-generator
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

---

## Build for production

```bash
npm run build
```

Output goes to `dist/`.

---

## Features

- Generate QR codes from any URL
- Add your name to the downloaded image
- Download as PNG or copy to clipboard
- Responsive — works on mobile, tablet, and laptop
- Single-page with Home, GitHub, and Contact sections

---

## Tech Stack

- [React 19](https://react.dev)
- [Vite 8](https://vitejs.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- [qrcode.react](https://github.com/zpao/qrcode.react)
